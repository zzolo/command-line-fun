Making the command line fun

Unfortunately this is only focused for Macs. Linux users can probably translate a lot of this, but Windows is widely different.

The command line (a.k.a. the terminal, or console, or shell, or CLI) is just a simplified, text-based interface into your computer.

The command line can be difficult to use and to learn, as it’s basically like learning a whole new system, and finding your way and getting help can be difficult.

Let’s make it a little easier, or at least a little more enjoyable. This guide will focus on getting the command line to look better, specifically in hopes to make it more usable, and along the way we’ll learn some basics and tips for using the command line.

What's the deal?

By default on a Mac, if you open up the Terminal application, you get this & a wall of white that is kind of intimidating, and it's not inviting you to get anything done.

And if you run a few commands, you get just get a wall of text, and it's very hard to read it and pull out information that you want.

The goal is to end up with something that is both fun and will actually help you get things done, and of course, we'll learn a bit about the mysterious command line along the way.

Commands

You will see commands to enter into the command line show like the following. If they are inline, they may show up like this command. Or they may separated out like this:

Money, money, money

Here and in other sites, you may see a $ symbol when describing putting a command into the command line. The $ is a generic symbol for your prompt; the prompt is just where you put in the command line. On the default Terminal application, it looks like this:


$ command --option

Other formatting

  • Terminal - This denotes an application.
  • Important term - This denotes an important term that will probably be used often.

Fonts

The first thing we want to do is install a new font. Specifically, we want a monospace font; a monospace font means that each character takes up the same width. This is extremely helpful for coding or the command line, as you can line things up vertically by spacing. MacOS comes up monospace fonts, but later on, we need a font that can support certain types of characters, so we’re gonna install some new fonts.

The prompt in the command line, is the set of characters that show up before you type in your command. Usually this show where you are in the filesystem, but it can be set to whatever, and can be very useful in giving you helpful info. We are going to install a prompt called Powerline. Powerline adds a lot of helpful information for most people and is very extensible. Powerline is a specific project, but there are many projects that are alternatives of Powerline that are called the same thing, so you can think of it as a general style of prompt.

For Powerline to work, we need fonts that support specific characters. Luckily, we have many options. The Powerline fonts project has many altered fonts that have the characters we need. You can use whatever one you want, but for consistency sake, let’s use Source Code Pro.

  1. Download: https://github.com/powerline/fonts/archive/master.zip
  2. Navigate to the file in Finder.
  3. Unzip it; you can do that by double-clicking it in Finder.
  4. Navigate into the new folder and into “SourceCodePro”
  5. Select all the “.off” files, right-click and choose “Open”
  6. This will open up a number of font dialog boxes. Choose “Install Font” on each one.
  7. Feel free to install other font sets, as you may prefer them.
